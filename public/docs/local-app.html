WLKT Mechanics 本地说明

本版本面向本地离线使用：界面资源、更新日志、说明页和计算逻辑都由本机加载，不依赖云端服务器。

安装为本地 App

  1. 在项目根目录执行 npm install 安装依赖。
  2. 执行 npm run app:build 生成离线构建。
  3. 执行 npm run app:preview，浏览器打开 http://127.0.0.1:4173。
  4. 在 Edge 或 Chrome 地址栏右侧选择“安装应用”，安装后即可从开始菜单或桌面启动。
首次安装需要本机预览服务提供安装入口。安装并打开过一次后，应用资源会被 service worker 缓存，可在离线状态下继续使用。

离线使用注意

  • 模型文件通过“保存项目”和“打开项目”在本地 JSON 文件中交换。
  • 分享链接只编码当前模型，不上传模型数据；在完全离线环境下建议优先使用项目 JSON 文件。
  • 本版本已移除 Google Analytics、Sentry、远程字体加载和周期性自动更新检查。
  • 荷载、弯矩、刚度、强度等输入支持 F、M、E、I、L、k、sigma、fy 等本地符号表达式。

适合材料力学教学的后续改进

  • 增加轴力、剪力、弯矩、挠度和转角的分步推导面板，显示平衡方程、边界条件和符号约定。
  • 加入典型题库模板，例如悬臂梁、简支梁、外伸梁、刚架、桁架、组合杆和温度荷载案例。
  • 为结果添加危险截面自动识别，标出最大正应力、最大剪应力、最大挠度和安全系数。
  • 提供单位一致性检查和量纲提示，减少 MPa、N、kN、mm、m 混用导致的输入错误。
  • 增加报告导出功能，将模型图、荷载、支反力、内力图和关键公式导出为 PDF。
